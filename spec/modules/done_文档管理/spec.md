# 文档管理

## 模块概述

文档管理为企业提供统一的文档存储、管理、共享、协作能力，支持文件夹管理、版本控制、权限控制和在线预览。

## 功能架构图

```text
文档管理
    ├── 文件夹管理
    │   ├── 新建/重命名/移动
    │   └── 复制/删除
    ├── 文件操作
    │   ├── 上传/下载
    │   ├── 在线预览
    │   └── 分享
    ├── 版本管理
    │   ├── 历史版本
    │   └── 版本回滚
    ├── 权限控制
    │   ├── 个人权限
    │   ├── 部门权限
    │   └── 角色权限
    └── 回收站
        ├── 删除保留
        └── 恢复/彻底删除
```

## 详细功能需求

### DM-1 文件夹管理

**功能说明**
支持多级文件夹的创建、重命名、移动、复制、删除。

**业务规则**
- 同一父文件夹下名称唯一；
- 删除文件夹时级联删除子文件（进入回收站）；
- 支持拖拽排序和移动。

### DM-2 文件上传/下载

**功能说明**
支持单文件和批量上传、断点续传、大文件分片。

**业务规则**
- 单文件大小限制由系统参数配置；
- 支持拖拽上传；
- 上传文件需进行类型和病毒扫描；
- 下载支持断点续传。

### DM-3 版本管理

**功能说明**
保留文件历史版本，支持版本回滚。

**业务规则**
- 默认保留最近 20 个版本；
- 支持版本备注；
- 回滚后生成新版本而非覆盖。

### DM-4 权限控制

**功能说明**
按个人、部门、角色设置文件/文件夹权限。

**业务规则**
- 权限类型：无权限、查看、编辑、管理；
- 子文件夹默认继承父文件夹权限；
- 文件夹权限和文件权限可单独设置。

### DM-5 在线预览

**功能说明**
无需下载即可预览常见文件格式。

**业务规则**
- 支持 Office、PDF、图片、文本、视频；
- 预览权限受文件权限控制；
- 大文件预览支持分页加载。

### DM-6 共享链接

**功能说明**
生成文件外部分享链接。

**业务规则**
- 链接可设置密码、有效期、访问次数；
- 支持撤销链接；
- 访问记录保存日志。

### DM-7 回收站

**功能说明**
被删除文件保留在回收站，支持恢复和彻底删除。

**业务规则**
- 保留 30 天，到期自动清理；
- 仅文件所有者或管理员可恢复；
- 彻底删除后不可恢复。

### DM-8 全文检索

**功能说明**
支持按文件标题和内容检索。

**业务规则**
- 支持关键词高亮；
- 按权限过滤搜索结果；
- 支持按文件类型、修改时间筛选。

## 用例描述

| 用例编号 | 用例名称 | 参与者 | 基本流程 |
|---|---|---|---|
| UC-DM-01 | 上传文件 | 普通员工 | 进入文档管理 → 选择文件夹 → 上传文件 |
| UC-DM-02 | 共享文件 | 普通员工 | 选择文件 → 生成共享链接 → 设置密码和有效期 |
| UC-DM-03 | 恢复删除文件 | 普通员工 | 进入回收站 → 选择文件 → 恢复 |

## 状态机/时序图描述

### 文件状态机

```text
[正常] --删除--> [回收站]
[回收站] --恢复--> [正常]
[回收站] --彻底删除--> [已删除]
[正常] --归档--> [已归档]
```

## 数据模型

| 实体 | 说明 | 核心字段 |
|---|---|---|
| oa_doc_folder | 文件夹 | id, parent_id, name, owner, permission, tenant_id |
| oa_doc_file | 文件主表 | id, folder_id, name, current_version, owner, size, tenant_id |
| oa_doc_version | 文件版本 | id, file_id, version_no, storage_path, size, remark, create_by |
| oa_doc_permission | 权限 | id, object_type, object_id, grantee_type, grantee_id, permission |
| oa_doc_share | 共享链接 | id, file_id, code, password, expire_time, max_count, access_count |
| oa_doc_recycle | 回收站 | id, object_type, object_id, name, delete_by, delete_time |
