# 知识库

## 模块概述

知识库用于沉淀企业知识资产，支持知识分类、词条编辑、全文检索、权限控制和用户互动，促进知识共享与复用。

## 详细功能需求

### KB-1 知识分类

**功能说明**
建立多级知识分类体系。

**业务规则**
- 分类层级最多 5 级；
- 分类编码唯一；
- 停用分类后其下词条不可见；
- 支持按分类设置默认权限。

### KB-2 知识词条

**功能说明**
创建和维护知识词条内容。

**业务规则**
- 词条标题唯一（同一分类下）；
- 支持草稿和发布状态；
- 发布后修改需重新发布或保存为新版本；
- 支持标签和附件。

### KB-3 富文本编辑

**功能说明**
提供富文本编辑器编辑词条内容。

**业务规则**
- 支持图片、视频、表格、代码块、超链接；
- 支持从文档导入内容；
- 编辑器内容自动保存草稿。

### KB-4 权限管理

**功能说明**
按分类或词条设置访问和编辑权限。

**业务规则**
- 权限类型：无权限、查看、编辑、管理；
- 分类权限可被词条权限覆盖；
- 支持按个人、部门、角色授权。

### KB-5 全文检索

**功能说明**
支持多维度知识检索。

**业务规则**
- 支持关键词高亮；
- 支持按分类、标签、作者、时间筛选；
- 搜索结果按相关性和热度排序。

### KB-6 收藏/评论

**功能说明**
用户可收藏和评论知识词条。

**业务规则**
- 用户可收藏常用词条到个人收藏夹；
- 评论支持富文本，需审核后显示（可配置）；
- 点赞/收藏数实时更新。

### KB-7 阅读统计

**功能说明**
统计词条的阅读量和互动数据。

**业务规则**
- 同一用户 24 小时内多次阅读只计一次；
- 统计维度：阅读量、点赞数、收藏数、评论数。

### KB-8 热门推荐

**功能说明**
基于热度向用户推荐知识。

**业务规则**
- 推荐算法综合考虑阅读量、点赞、收藏、时间衰减；
- 支持管理员手动置顶推荐。

## 用例描述

| 用例编号 | 用例名称 | 参与者 | 基本流程 |
|---|---|---|---|
| UC-KB-01 | 创建词条 | 知识管理员 | 选择分类 → 填写内容 → 设置权限 → 发布 |
| UC-KB-02 | 检索知识 | 普通员工 | 输入关键词 → 筛选分类 → 查看词条 |
| UC-KB-03 | 收藏词条 | 普通员工 | 查看词条 → 点击收藏 → 在个人收藏夹查看 |

## 数据模型

| 实体 | 说明 | 核心字段 |
|---|---|---|
| oa_kb_category | 知识分类 | id, parent_id, name, code, permission, status, tenant_id |
| oa_kb_article | 知识词条 | id, category_id, title, summary, content, tags, author, status, version, tenant_id |
| oa_kb_version | 词条版本 | id, article_id, version_no, content, change_log, create_by |
| oa_kb_tag | 标签 | id, name, tenant_id |
| oa_kb_article_tag | 词条标签关系 | id, article_id, tag_id |
| oa_kb_favorite | 收藏 | id, user_id, article_id, tenant_id |
| oa_kb_comment | 评论 | id, article_id, user_id, content, status, tenant_id |
| oa_kb_read_log | 阅读记录 | id, article_id, user_id, read_time, tenant_id |
