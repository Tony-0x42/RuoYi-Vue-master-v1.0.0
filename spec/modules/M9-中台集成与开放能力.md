# M9 中台集成与开放能力

## 模块概述

本模块面向外部系统与中台化运营，提供标准 REST API、状态变更回调、事件订阅、Webhook 节点、连接器框架、数据映射引擎、单点登录、API 网关、多租户隔离等能力。

## 详细功能需求

### M9.1 流程发起 API

**功能说明**
标准 REST API 供外部系统调用发起流程实例，支持同步返回实例 ID 或异步回调。

**业务规则**
- 请求需携带租户 ID、流程定义 Key、发起人、业务标识、表单数据；
- 同步返回包含 instanceId、currentNode、taskList；
- 异步发起返回请求 ID，完成后通过回调通知。

**前置条件**
外部系统已完成接入认证。

**后置条件**
流程实例创建并返回结果或进入回调队列。

**异常处理**
- 流程定义不存在/未发布：返回 404 与错误码；
- 参数校验失败：返回 400 与字段级错误；
- 并发限流：返回 429。

**界面原型描述**
开发者门户展示接口文档与在线调试工具。

### M9.2 流程状态查询 API

**功能说明**
按实例 ID 查询当前节点、审批历史记录、流程状态、节点耗时。

**业务规则**
- 查询结果按权限过滤；
- 支持返回 BPMN 流程图 XML；
- 节点耗时包含进入时间、处理时间、处理人。

**前置条件**
实例 ID 存在。

**后置条件**
返回实例状态与历史记录。

**异常处理**
- 无权查看：返回 403。

**界面原型描述**
开发者门户展示请求/响应示例。

### M9.3 流程操作 API

**功能说明**
外部系统代表用户执行同意、拒绝、转交、退回、终止等审批操作，需身份鉴权。

**业务规则**
- 操作请求需携带用户身份标识；
- 操作需校验任务处理人/代理权限；
- 批量操作支持一次性提交多个任务。

**前置条件**
任务处于待处理状态。

**后置条件**
任务按操作更新状态，流程继续流转。

**异常处理**
- 任务已被处理：返回 409 冲突；
- 无权限：返回 403。

**界面原型描述**
API 文档按操作类型分组展示。

### M9.4 状态变更回调

**功能说明**
流程状态发生变更时主动回调外部系统（发起/审批中/完成/拒绝/终止），支持重试与幂等。

**业务规则**
- 回调 URL 在流程定义或外部系统注册时配置；
- 回调请求包含幂等 ID（idempotencyKey）；
- 失败回调按指数退避重试，最多 5 次。

**前置条件**
外部系统已注册回调地址。

**后置条件**
状态变更事件进入回调队列并推送。

**异常处理**
- 回调连续失败：标记外部系统回调异常并通知管理员。

**界面原型描述**
「回调管理」页面展示回调日志与重试状态。

### M9.5 事件订阅机制

**功能说明**
外部系统可订阅指定流程定义或节点的特定事件（如“请假审批完成”），事件触发时推送。

**业务规则**
- 订阅支持按流程 Key、节点 ID、事件类型过滤；
- 支持 WebSocket、Webhook、MQ 三种推送方式；
- 订阅关系支持启用/禁用。

**前置条件**
外部系统已创建订阅。

**后置条件**
匹配事件触发后推送到订阅方。

**异常处理**
- 订阅规则非法：创建时校验失败。

**界面原型描述**
「事件订阅」页面管理订阅规则。

### M9.6 Webhook 节点

**功能说明**
流程节点中配置外部 HTTP 调用（GET/POST），支持同步等待响应（影响分支）或异步触发。

**业务规则**
- 同步 Webhook 超时时间默认 30 秒；
- 响应结果可写入流程变量；
- 失败分支可配置重试或异常处理。

**前置条件**
流程设计器中已添加 Webhook 节点。

**后置条件**
流程运行到节点时发起 HTTP 请求。

**异常处理**
- 网络超时：按失败分支处理；
- 响应格式错误：记录日志。

**界面原型描述**
Webhook 节点属性面板配置 URL、方法、请求头、请求体、超时、失败策略。

### M9.7 连接器框架

**功能说明**
预置常见系统连接器（ERP、CRM、HR、财务系统），支持自定义连接器开发（配置化或代码）。

**业务规则**
- 连接器封装认证、请求、错误处理；
- 自定义连接器遵循统一接口规范；
- 连接器版本独立管理。

**前置条件**
已安装并配置连接器。

**后置条件**
流程可通过连接器节点调用外部系统。

**异常处理**
- 连接器未安装：提示“连接器不存在”。

**界面原型描述**
「连接器市场」展示已安装与可安装连接器。

### M9.8 数据映射引擎

**功能说明**
流程表单字段与外部系统 API 参数的映射配置，支持字段转换、常量填充、表达式计算。

**业务规则**
- 映射支持一对一、一对多、多对一；
- 转换函数支持字符串、日期、数值等常用函数；
- 映射配置随流程版本管理。

**前置条件**
存在 Webhook/连接器节点。

**后置条件**
运行时按映射生成请求参数或回填响应数据。

**异常处理**
- 必填映射字段缺失：请求失败并记录。

**界面原型描述**
映射配置页面左侧表单字段，右侧目标字段，支持拖拽绑定。

### M9.9 单点登录集成

**功能说明**
支持 SSO 统一认证、JWT Token 鉴权、OAuth 2.0 / OIDC 标准协议。

**业务规则**
- 支持企业微信/钉钉/飞书/AD/LDAP 等身份源；
- Token 有效期可配置；
- 支持多租户独立身份源配置。

**前置条件**
已配置身份源参数。

**后置条件**
用户可通过 SSO 登录并获取 Token。

**异常处理**
- 身份源不可用：提示并支持本地账号登录。

**界面原型描述**
登录页展示企业微信/钉钉/账号密码登录入口。

### M9.10 API 网关

**功能说明**
统一 API 入口管理、请求限流、访问鉴权、请求日志、性能监控、版本兼容。

**业务规则**
- 限流策略支持 QPS、并发数、令牌桶；
- API 版本通过 URL 路径或 Header 区分；
- 请求日志保留 90 天。

**前置条件**
API 网关已部署。

**后置条件**
外部请求经网关路由到后端服务。

**异常处理**
- 触发限流：返回 429；
- 鉴权失败：返回 401。

**界面原型描述**
网关控制台展示 API 列表、流量、告警。

### M9.11 多租户隔离

**功能说明**
租户级流程模板隔离、租户级数据隔离、租户级管理员、租户级组织架构同步。

**业务规则**
- 所有业务表包含 tenant_id 字段；
- 租户管理员只能管理本租户数据；
- 超级管理员可跨租户查看与操作。

**前置条件**
租户已创建。

**后置条件**
租户内数据与配置相互隔离。

**异常处理**
- 租户 ID 缺失：返回 400。

**界面原型描述**
超级管理员后台展示租户列表，支持新增/编辑/禁用租户。

## 状态机/时序图描述

### 外部系统发起流程时序

1. 外部系统获取 AccessToken；
2. 调用 API 网关；
3. 网关鉴权与限流；
4. 流程引擎创建实例；
5. 返回实例 ID 或异步回调；
6. 状态变更时推送回调。
