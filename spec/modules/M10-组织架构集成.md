# M10 组织架构集成

## 模块概述

组织架构集成模块实现人员、部门、角色、主管链、人员标签、兼职/兼岗、离职转交、组织变更适配等能力，为审批人解析提供统一、实时、灵活的组织数据支撑。

## 详细功能需求

### M10.1 人员数据同步

**功能说明**
与 HR 系统/组织架构中心实时/定时同步用户数据，支持增量同步与全量同步。

**业务规则**
- 支持 WebService/REST/MQ 等多种同步方式；
- 增量同步按更新时间戳或事件触发；
- 全量同步建议每周一次；
- 同步失败记录日志并告警。

**前置条件**
已配置 HR 系统连接参数。

**后置条件**
组织数据与中台保持一致。

**异常处理**
- 同步失败：保留旧数据，进入重试队列。

**界面原型描述**
「组织同步」页面展示同步方式、定时任务、最近同步日志。

### M10.2 部门树集成

**功能说明**
支持多级部门结构、虚拟部门/项目组、矩阵式组织、部门排序与层级展示。

**业务规则**
- 部门树支持无限层级；
- 虚拟部门不参与编制汇报，仅用于项目/任务；
- 矩阵式组织支持一人多汇报线。

**前置条件**
部门数据已同步。

**后置条件**
部门选择器、主管查找等依赖部门树。

**异常处理**
- 部门循环依赖：同步时检测并拒绝。

**界面原型描述**
组织架构页面以树形展示部门，支持展开/折叠/搜索。

### M10.3 角色体系

**功能说明**
岗位角色（如“财务经理”）、业务角色（如“报销审批人”）、流程角色（流程内临时角色）三级体系。

**业务规则**
- 岗位角色与组织架构绑定；
- 业务角色可跨部门授予；
- 流程角色仅在流程实例内有效。

**前置条件**
角色数据已维护。

**后置条件**
审批节点选择角色来源时可选择对应角色。

**异常处理**
- 角色下无人员：任务挂起并告警。

**界面原型描述**
「角色管理」页面按三级体系维护角色与人员关系。

### M10.4 主管查找规则

**功能说明**
支持直接上级、第 N 级上级、直到某级、部门负责人链、虚线上级、项目 Leader 查找。

**业务规则**
- 主管链基于组织汇报线实时计算；
- 虚线上级需单独维护虚线关系；
- 项目 Leader 通过项目角色识别。

**前置条件**
组织架构与汇报关系完整。

**后置条件**
解析出具体主管用户 ID。

**异常处理**
- 主管链中断：返回可找到的最高主管或转交配置接替人。

**界面原型描述**
审批节点配置「主管查找」下拉规则。

### M10.5 动态审批人计算

**功能说明**
根据表单字段值、发起人信息、运行时数据动态计算审批人。

**业务规则**
- 表达式可引用表单字段、流程变量、发起人部门/职级；
- 返回结果为用户/角色/部门；
- 计算时机为任务创建前。

**前置条件**
审批节点配置动态表达式来源。

**后置条件**
任务创建时按计算结果分配。

**异常处理**
- 计算结果为空：任务挂起。

**界面原型描述**
表达式编辑器提供字段、函数、组织函数提示。

### M10.6 人员标签

**功能说明**
给用户打标签（如“财务专家”“合规专员”），审批人选择时按标签筛选。

**业务规则**
- 标签可手动维护或通过接口导入；
- 一个用户可有多个标签；
- 标签支持按租户隔离。

**前置条件**
标签数据已维护。

**后置条件**
人员选择器支持按标签筛选。

**异常处理**
- 标签不存在：提示并忽略。

**界面原型描述**
用户详情页维护标签；人员选择器支持标签搜索。

### M10.7 兼职/兼岗支持

**功能说明**
一人多岗场景下的主岗/兼岗识别，流程中按业务类型匹配对应岗位的主管。

**业务规则**
- 用户主岗唯一，兼岗可多个；
- 流程中可选择按主岗或指定兼岗解析主管；
- 兼岗信息随组织同步。

**前置条件**
用户存在主岗/兼岗数据。

**后置条件**
主管查找按指定岗位类型计算。

**异常处理**
- 兼岗冲突：以主岗为准。

**界面原型描述**
用户详情页展示主岗兼岗列表。

### M10.8 离职自动转交

**功能说明**
审批人离职/禁用后，系统自动将其待办转交给其上级或预先指定的接替人。

**业务规则**
- 人员状态变更为离职/禁用后触发；
- 转交范围包括待处理任务、公共任务池任务；
- 转交记录留痕。

**前置条件**
用户状态变更。

**后置条件**
待办任务转交给接替人。

**异常处理**
- 未配置接替人：转交给直属上级。

**界面原型描述**
「人员管理」页面配置接替人，离职流程自动触发。

### M10.9 组织变更适配

**功能说明**
组织架构调整（部门合并/拆分/人员调岗）后，运行中流程的审批人动态适配。

**业务规则**
- 已创建任务按原审批人保留，新任务按新组织架构解析；
- 部门合并/拆分可配置映射关系；
- 变更适配记录审计日志。

**前置条件**
组织架构发生变更。

**后置条件**
运行中流程后续节点按新组织架构执行。

**异常处理**
- 映射缺失：按新部门负责人兜底。

**界面原型描述**
「组织变更适配」页面配置部门映射与生效时间。

## 状态机/时序图描述

### 主管查找时序

1. 任务创建请求；
2. 解析审批人来源为「第 N 级上级」；
3. 查询发起人组织汇报线；
4. 向上递归 N 级；
5. 返回用户 ID；
6. 若中断则按兜底规则处理。
