# M2 表单设计器

## 模块概述

表单设计器为流程提供动态数据收集能力，支持基础输入、选择类、组织类、高级、业务、布局六大类组件，并提供组件属性配置、校验规则引擎、表单联动机制、计算公式引擎、样式与适配能力。

```text
└────────────────────────────────────────────┘
    │
    ▼
┌────────────────────────────────────────────┐
│  表单定义服务：字段元数据 + 渲染配置 + 校验规则 │
│  联动规则 │ 计算公式 │ 权限规则              │
└────────────────────────────────────────────┘
    │
    ▼
┌────────────────────────────────────────────┐
│  运行时渲染引擎（PC/移动端）                  │
└────────────────────────────────────────────┘
```

## 详细功能需求

### M2.1 基础输入组件

**功能说明**
提供单行文本、多行文本、数字输入、金额输入、日期选择、日期范围、时间选择组件。

**业务规则**
- 金额组件自动千分位展示，支持切换大小写显示；
- 数字组件支持整数/小数精度配置；
- 日期范围组件返回 `[startDate, endDate]`；
- 字段编码全局唯一，使用英文+下划线命名。

**前置条件**
表单设计器已加载。

**后置条件**
组件可拖拽到画布并配置属性。

**异常处理**
- 字段编码重复：提示“字段编码已存在”；
- 金额精度非法：限制小数位 0~4。

**界面原型描述**
左侧组件库「基础字段」分组，拖拽到画布后右侧属性面板显示字段名称、编码、默认值、占位提示、必填等。

### M2.2 选择类组件

**功能说明**
提供单选框、多选框、下拉选择（单/多选）、级联选择、树形选择、滑动开关。

**业务规则**
- 选项来源支持静态配置、数据字典、外部接口；
- 级联选择支持 2~5 级；
- 树形选择支持单选/多选、父子级联选中。

**前置条件**
已创建表单。

**后置条件**
选项配置可被联动规则引用。

**异常处理**
- 外部接口拉取失败：保留上次选项或显示空列表。

**界面原型描述**
属性面板「选项」Tab 支持手动输入、字典选择、接口选择。

### M2.3 组织类组件

**功能说明**
成员选择（单/多选）、部门选择、角色选择，支持组织架构树实时搜索。

**业务规则**
- 成员选择结果存储用户 ID 数组；
- 部门选择结果存储部门 ID 数组；
- 支持按部门/角色范围限定可选人员。

**前置条件**
已同步组织架构数据。

**后置条件**
运行时显示组织架构树并支持搜索。

**异常处理**
- 组织架构未同步：提示“组织架构数据缺失，请联系管理员”。

**界面原型描述**
点击选择框弹出组织架构树弹窗，顶部搜索框支持按姓名/工号/部门搜索。

### M2.4 高级组件

**功能说明**
附件上传（多文件/限制类型/限制大小）、图片上传（预览/裁剪）、手写签名板、富文本编辑器、定位组件。

**业务规则**
- 附件大小默认最大 50 MB，类型限制可通过配置调整；
- 图片上传支持 JPG/PNG，最大 10 MB；
- 手写签名生成 PNG 图片并作为附件存储；
- 定位组件返回经纬度与地址文本。

**前置条件**
已配置文件存储服务（OSS/MinIO/本地）。

**后置条件**
运行时组件可正常上传与展示。

**异常处理**
- 文件超限时提示“文件大小超过 X MB”；
- 上传失败时保留错误日志。

**界面原型描述**
附件组件显示上传列表，支持下载、删除、预览；签名组件弹窗提供手写板。

### M2.5 业务组件

**功能说明**
关联表单（跨表单数据引用）、子表单/明细表（动态增删行）、流水号（自定义编码规则）、计算公式、关联数据填充。

**业务规则**
- 子表单每行字段独立，支持行内校验；
- 流水号规则支持日期前缀、随机码、自增序号；
- 关联数据填充可配置来源字段到目标字段的映射。

**前置条件**
已存在被关联的表单或数据接口。

**后置条件**
运行时组件可动态渲染与计算。

**异常处理**
- 关联表单不存在：提示“关联表单已删除或不可用”。

**界面原型描述**
子表单以表格形式展示，支持“新增一行”；流水号组件只读显示生成结果。

### M2.6 布局组件

**功能说明**
标题、分割线、Tabs 页签、栅格布局（24 列）、卡片容器，支持响应式布局。

**业务规则**
- 栅格列数之和必须为 24；
- Tabs 页签支持配置默认激活项；
- 卡片容器可嵌套字段组件。

**前置条件**
表单画布已初始化。

**后置条件**
布局配置影响运行时渲染结构。

**异常处理**
- 栅格列数超过 24 时自动归位提示。

**界面原型描述**
画布以虚线框展示栅格，拖拽组件到栅格单元格内。

### M2.7 组件属性配置

**功能说明**
配置字段名称、字段编码（数据库列名）、占位提示文本、默认值（静态/动态/表达式）、是否必填、自定义校验规则。

**业务规则**
- 字段名称必填，2~50 字符；
- 字段编码必填，英文/数字/下划线，不能以数字开头；
- 默认值支持静态值、流程变量、表达式三种类型。

**前置条件**
已选中画布上的组件。

**后置条件**
属性变更实时保存到表单定义。

**异常处理**
- 非法字段编码：高亮输入框并提示规则。

**界面原型描述**
右侧属性面板按「基础」「高级」「校验」「联动」分组展示配置项。

### M2.8 校验规则引擎

**功能说明**
支持必填校验、最小/最大长度、正则表达式匹配、数值范围、邮箱/手机格式、唯一性校验、自定义函数校验。

**业务规则**
- 必填校验在提交时触发；
- 自定义校验函数支持 JavaScript 片段；
- 唯一性校验需指定校验范围（流程实例内/全局）。

**前置条件**
组件已配置字段编码。

**后置条件**
运行时按规则校验输入。

**异常处理**
- 自定义函数语法错误：设计时提示“校验函数存在语法错误”。

**界面原型描述**
属性面板「校验」Tab 提供规则列表，可新增/删除/排序规则。

### M2.9 表单联动机制

**功能说明**
字段显隐联动（A 字段值控制 B 字段显示/隐藏）、选项联动、数据自动填充、级联过滤、子表单联动。

**业务规则**
- 联动条件支持等于/不等于/包含/不包含等；
- 多个条件可配置 AND/OR 组合；
- 联动触发时机包括值变化、表单加载。

**前置条件**
表单中至少存在两个可关联字段。

**后置条件**
运行时根据条件动态更新目标字段状态。

**异常处理**
- 联动目标字段被删除：自动清除联动规则。

**界面原型描述**
「联动」Tab 可视化配置“当 A 满足 X 时，B 显示/隐藏/填充/过滤”。

### M2.10 计算公式引擎

**功能说明**
字段间数学运算（加减乘除、取模）、字符串拼接、日期差值计算、条件公式（IF/THEN/ELSE）、聚合函数（SUM/COUNT/AVG）。

**业务规则**
- 公式表达式语法使用类 Excel 语法，如 `SUM(子表单.金额)`；
- 计算触发时机：字段值变化、表单加载、提交前；
- 除零错误时结果为空并记录日志。

**前置条件**
存在可参与计算的字段。

**后置条件**
计算结果实时回填到目标字段。

**异常处理**
- 公式引用不存在字段：提示“公式中引用字段不存在”。

**界面原型描述**
属性面板「计算公式」输入框，提供字段选择与函数提示。

### M2.11 表单样式与适配

**功能说明**
主题配色、标签位置（上/左/右）、组件宽度调节、移动端自适应渲染、暗黑模式支持。

**业务规则**
- 标签位置默认左对齐；
- 移动端字段自动单列排列；
- 暗黑模式跟随系统或用户手动切换。

**前置条件**
表单已保存。

**后置条件**
运行时按样式配置渲染。

**异常处理**
- 主题配置缺失时使用默认主题。

**界面原型描述**
设计器顶部提供「PC 预览」「移动端预览」「主题切换」按钮。
