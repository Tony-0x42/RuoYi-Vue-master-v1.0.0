# 资产管理产品需求文档

## 1. 背景与目标

### 1.1 背景

企业固定资产数量多、分布广，传统台账管理容易遗漏，盘点困难，资产流失风险高。

### 1.2 目标

建立资产全生命周期管理系统，实现资产入库、领用、调拨、维修、盘点、报废的线上化管理，确保账实一致。

## 2. 目标用户

| 用户角色 | 主要诉求 |
|---|---|
| 资产管理员 | 管理资产台账、组织盘点、处理报废 |
| 普通员工 | 申请领用资产、查看自己使用的资产 |
| 行政人员 | 管理办公资产、统计资产使用情况 |
| 财务人员 | 掌握资产价值、折旧和处置情况 |

## 3. 用户故事

- **作为** 资产管理员，**我希望** 扫码查看资产信息，**以便** 快速盘点。
- **作为** 普通员工，**我希望** 在线申请领用资产，**以便** 快速获得办公设备。
- **作为** 财务人员，**我希望** 查看资产台账和报废记录，**以便** 准确核算资产价值。

## 4. 业务流程

```text
资产入库
  └── 资产台账
        ├── 领用申请 → 审批 → 在用
        ├── 调拨申请 → 审批 → 变更使用人
        ├── 维修申请 → 维修完成 → 恢复使用
        ├── 盘点任务 → 扫码盘点 → 处理差异
        └── 报废申请 → 审批 → 已报废
```

## 5. 功能详述

### 5.1 资产分类

- 支持多级分类，如电子设备、办公家具。
- 分类支持启用/停用。

### 5.2 资产台账

- 记录资产编码、名称、规格、型号、价值、使用人、状态。
- 展示资产全生命周期履历。

### 5.3 资产入库

- 支持单条/批量入库。
- 自动生成资产编码和二维码。

### 5.4 资产领用/归还

- 在线申请领用，审批通过后更新使用人。
- 归还时确认资产状态。

### 5.5 资产调拨

- 资产在部门/人员间转移，需审批。

### 5.6 资产维修

- 记录维修申请、维修费用、维修商。

### 5.7 资产盘点

- 创建盘点任务，扫码盘点。
- 自动生成盘点差异报告。

### 5.8 资产报废

- 报废申请和审批。
- 记录报废原因和处置方式。

## 6. 原型描述

### 资产列表页

- 顶部：搜索、分类筛选、状态筛选。
- 中部：资产列表，展示编码、名称、使用人、状态。

### 资产详情页

- 基本信息、使用记录、维修记录、盘点记录、报废记录。

### 盘点页面

- 盘点任务列表、扫码盘点入口、盘点差异报告。

## 7. 验收标准

- [ ] 支持资产分类和台账管理。
- [ ] 支持资产入库并生成二维码。
- [ ] 支持领用、归还、调拨、维修、报废全流程审批。
- [ ] 支持资产盘点和差异处理。
- [ ] 支持扫码查询资产详情。
- [ ] 支持资产统计报表。

## 8. 依赖与风险

### 依赖

- 流程引擎；
- 二维码生成和扫码能力；
- 组织架构；
- 打印设备（标签打印）。

### 风险

- 资产标签损坏可能导致盘点困难，需支持手动输入编码查询。
