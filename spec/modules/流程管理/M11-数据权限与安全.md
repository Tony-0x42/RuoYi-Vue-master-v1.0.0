# M11 数据权限与安全

## 模块概述

本模块保障流程数据的访问安全与合规，覆盖字段级权限、节点差异化表单、数据范围权限、操作权限、附件/打印权限、数据脱敏、日志审计、数据加密等能力。

## 详细功能需求

### M11.1 字段级权限

**功能说明**
每个流程节点独立配置字段可见、可编辑、隐藏、必填四态权限。

**业务规则**
- 可见：字段显示但不可编辑；
- 可编辑：字段显示且可编辑；
- 隐藏：字段不渲染且不提交；
- 必填：字段必须填写，仅在可编辑态生效；
- 发起人节点默认所有业务字段可编辑。

**前置条件**
表单已设计完成。

**后置条件**
运行时根据当前节点与用户权限渲染字段。

**异常处理**
- 必填字段被隐藏：校验时排除该字段必填。

**界面原型描述**
「字段权限」矩阵页面，行是字段，列是节点，单元格选择权限态。

### M11.2 节点差异化表单

**功能说明**
不同节点可展示不同表单布局、不同字段集合，实现发起/审批/归档的表单差异化。

**业务规则**
- 支持为不同节点绑定不同表单或同一表单的不同视图；
- 归档节点可展示打印/签章区域；
- 差异配置随流程版本管理。

**前置条件**
存在多个表单视图或布局配置。

**后置条件**
运行时按节点加载对应表单视图。

**异常处理**
- 节点未绑定视图：使用默认视图。

**界面原型描述**
节点属性「表单视图」下拉选择已定义的视图。

### M11.3 数据范围权限

**功能说明**
按部门、角色、用户组限制可查看的流程数据范围（如仅看本部门发起的流程）。

**业务规则**
- 范围规则支持：本人、本部门、本部门及下级、指定部门、全部；
- 可按流程定义单独配置；
- 超级管理员不受限制。

**前置条件**
流程定义已发布。

**后置条件**
流程列表与详情按范围过滤。

**异常处理**
- 规则冲突：按最小范围原则。

**界面原型描述**
「流程权限」页面配置数据范围规则。

### M11.4 操作权限控制

**功能说明**
查看、编辑、删除、终止、转办、导出等操作按角色进行细粒度授权。

**业务规则**
- 操作权限支持按角色、部门、用户、流程分类配置；
- 管理员可强制覆盖普通权限；
- 操作按钮根据权限动态显示。

**前置条件**
角色与权限数据已维护。

**后置条件**
用户只能看到被授权的操作。

**异常处理**
- 越权操作：返回 403。

**界面原型描述**
「权限矩阵」页面勾选角色与操作对应关系。

### M11.5 附件权限

**功能说明**
附件的查看、下载、上传权限独立控制，敏感附件支持水印与防泄密。

**业务规则**
- 附件权限可配置到流程/节点/字段级；
- 敏感附件下载时自动添加用户 ID 水印；
- 禁止转发/截屏策略在移动端通过 SDK 控制。

**前置条件**
表单中存在附件组件。

**后置条件**
运行时按权限控制附件操作。

**异常处理**
- 无下载权限：按钮隐藏或点击提示。

**界面原型描述**
附件组件右侧显示权限说明；下载文件带水印。

### M11.6 打印权限

**功能说明**
是否允许打印流程表单、打印模板选择、打印时自动添加水印（含用户信息/时间戳）。

**业务规则**
- 打印权限按角色/流程/节点配置；
- 打印水印包含用户姓名、工号、打印时间；
- 支持禁止打印的字段单独配置。

**前置条件**
存在打印模板。

**后置条件**
打印输出按权限与水印渲染。

**异常处理**
- 无打印权限：页面打印按钮禁用。

**界面原型描述**
流程详情页「打印」按钮按权限显示，打印预览展示水印。

### M11.7 数据脱敏

**功能说明**
敏感字段（手机号、身份证号、银行卡号）按规则脱敏显示，仅授权人可见完整内容。

**业务规则**
- 默认规则：手机号 `138****1234`、身份证 `110101********1234`、银行卡 `**** **** **** 1234`；
- 脱敏规则可自定义；
- 授权人点击「查看完整」需二次认证并留痕。

**前置条件**
字段标记为敏感字段。

**后置条件**
运行时按规则显示脱敏内容。

**异常处理**
- 数据格式不符合规则：显示为「***」。

**界面原型描述**
敏感字段默认显示脱敏内容，授权用户点击眼睛图标查看完整。

### M11.8 日志审计

**功能说明**
记录操作日志、审批日志、字段变更日志、登录日志，支持检索与导出。

**业务规则**
- 操作日志记录操作人、时间、IP、操作类型、对象、结果；
- 审批日志记录每次审批的操作、意见、时间、处理人；
- 字段变更日志记录修改前后值；
- 日志保留期限可配置，默认 3 年。

**前置条件**
系统启用日志记录。

**后置条件**
日志写入审计库，可检索导出。

**异常处理**
- 日志写入失败：写入本地文件并告警。

**界面原型描述**
「审计中心」页面提供多维度查询与导出。

### M11.9 数据加密

**功能说明**
传输层 TLS 加密、存储层 AES 加密、敏感字段独立加密、密钥轮换机制。

**业务规则**
- 所有外部通信使用 TLS 1.2+；
- 数据库敏感字段使用 AES-256 加密存储；
- 密钥由 KMS 统一管理，支持定期轮换；
- 加密字段在内存中解密后按需使用。

**前置条件**
已配置 KMS 与加密策略。

**后置条件**
敏感数据全生命周期加密。

**异常处理**
- 密钥不可用：服务降级并告警，禁止读取敏感数据。

**界面原型描述**
「安全策略」页面配置加密字段与密钥轮换周期。

## 状态机/时序图描述

### 表单权限渲染时序

1. 用户进入审批页面；
2. 获取当前节点与用户角色；
3. 查询字段权限矩阵；
4. 按权限态渲染字段；
5. 提交时按权限过滤不可编辑字段。
