# 七、集成需求

## 7.1 集成总体架构

```text
┌─────────────────────────────────────────────────────────────┐
│                     统一流程引擎中台                         │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ 流程设计器   │  │ 流程运行时   │  │ 监控与运营           │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
└──────────┬───────────────────────┬──────────────────────────┘
           │                       │
    ┌──────▼──────┐        ┌───────▼────────┐
    │   API 网关   │        │   事件总线      │
    └──────┬──────┘        └───────┬────────┘
           │                       │
   ┌───────┼───────┐      ┌────────┼────────┐
   ▼       ▼       ▼      ▼        ▼        ▼
 OA/ERP  CRM   HR    SSO  MQ    Email   钉钉/企业微信/飞书
```

## 7.2 与 HR 系统集成

### 数据流

- **同步方向**：HR → 流程中台；
- **数据内容**：用户、部门、岗位、汇报关系、兼职、离职状态、人员标签；
- **同步方式**：
  - 定时全量同步（建议每周一次）；
  - 增量同步（按更新时间戳或 HR 事件推送）；
  - 组织架构变更事件通过 MQ 实时同步。

### 接口要求

- HR 提供 REST/WebService 接口或推送 MQ 消息；
- 中台提供接收接口与数据结构约束；
- 同步失败进入重试队列并告警。

## 7.3 与 ERP/CRM/财务系统集成

### 数据流

- **发起流程**：ERP 调用中台 API 发起流程；
- **状态回调**：中台通过回调或事件订阅通知 ERP 流程状态；
- **数据回填**：流程中通过 Webhook/连接器调用 ERP 接口写入审批结果。

### 连接器示例

| 连接器 | 功能 | 典型场景 |
|---|---|---|
| ERP 连接器 | 读取订单/采购数据、写入审批结果 | 采购申请审批 |
| CRM 连接器 | 读取客户/合同数据 | 合同审批 |
| 财务连接器 | 读取预算、写入凭证 | 报销审批 |

## 7.4 与 SSO/身份源集成

### 数据流

- 用户访问中台或业务系统时，跳转 SSO 登录；
- SSO 返回 Token 与用户身份；
- 中台校验 Token 并解析用户、部门、角色信息。

### 支持协议

- OAuth 2.0 / OIDC；
- SAML 2.0（可选）；
- LDAP/AD；
- 企业微信/钉钉/飞书扫码登录。

## 7.5 与消息/通知平台集成

### 数据流

- 流程事件（待办、通过、拒绝、催办）进入消息中心；
- 消息中心按用户偏好与模板通过多渠道发送；
- 支持站内信、邮件、短信、企业微信、钉钉、飞书。

### 模板变量

| 变量 | 说明 |
|---|---|
| {{发起人}} | 流程发起人姓名 |
| {{流程名称}} | 流程定义名称 |
| {{节点名称}} | 当前节点名称 |
| {{处理人}} | 任务处理人 |
| {{审批意见}} | 审批意见 |
| {{链接}} | 流程详情链接 |

## 7.6 与文件存储/OSS 集成

- 附件、签名图片、打印模板、归档文件存储到对象存储；
- 支持 MinIO、阿里云 OSS、AWS S3；
- 文件访问 URL 带签名校验与过期时间。

## 7.7 集成安全

- 所有跨系统通信使用 HTTPS；
- 回调请求签名验证；
- API 调用基于 OAuth 2.0 Client Credentials 或 JWT；
- 敏感数据传输加密；
- 集成日志完整审计。
